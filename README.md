# Color-based-target-tracking-using-MATLAB
## Features
- Real-time tracking
- Target tracking based on PID algorithm
- Accurately track objects in red, green and blue

## Demo
- **Pink Cross:** Center coordinate of the object
- **Blue Dot:** Real-time aiming point
